# 智慧閱讀平臺・教育雲與線上資料庫介接 釐清說明

致建置廠商**凌網資訊**的線上簡報 — 2026.06.22（民國 115 年）11:00 線上會議用。

## 內容

就「智慧閱讀平臺承接線上資料庫」所需的 **教育雲（教育部 OpenID）身分驗證** 介接，向凌網釐清掌握程度，並說明本中心可提供的協助：

1. 想了解凌網對 **教育雲介接** 的掌握程度（線上資料庫 `getAuthentication` loginMode＝16 OPEN_ID 會用到）。
2. 凌網已有 **臺北市單一身分驗證 SSO（loginMode＝1）** 經驗，教育雲可望為延伸。
3. 若教育局能取得線上資料庫原始碼，本中心將改寫消化後提供凌網參考（條件式）。
4. 本中心其他系統（ailms 等）已做過教育雲介接，可整理經驗提供。

## 介接文件

線上資料庫介接規格 V1.5、教育雲（單一身分驗證）介接手冊等介接文件，**不放公開網路**，由本中心於會後另以 email／限閱方式提供凌網。
